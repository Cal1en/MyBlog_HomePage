通常情况下，我们不会直接在材质球面板中检索后处理的`Shader`，因此我们通常会将`Shader`的路径设置为`"Hidden/"`目录下，这样该`Shader`就会在材质球面板中被隐藏。

# 后处理`Shader`模版
Unity自带了后处理`Shader`的模板，同时由于后处理效果基本用不到顶点着色器，因此我们可以直接调用Unity自带的 **`vert_img`顶点着色器函数** 以及 **`v2f_img`结构体** 来代替。

```csharp
Shader "Hidden/ColorAdjustment"
{
    Properties
    {
        _MainTex ("Texture", 2D) = "white" {}
    }
    SubShader
    {
        // No culling or depth
        Cull Off ZWrite Off ZTest Always

        Pass
        {
            CGPROGRAM
            #pragma vertex vert_img
            #pragma fragment frag

            #include "UnityCG.cginc"

            sampler2D _MainTex;

            half4 frag (v2f_img i) : SV_Target
            {
                half4 col = tex2D(_MainTex, i.uv);
                return col;
            }
            ENDCG
        }
    }
}
```

# 亮度、饱和度、对比度调整
Unity自带了计算灰度值的函数`Luminance()`，其计算公式为

`lum = dot(color.rgb, float3(0.22, 0.707, 0.071));`

```csharp
Shader "Hidden/CS05/ColorAdjustment"
{
    Properties
    {
        [HideInInspector]_MainTex ("Texture", 2D) = "white" {}

        //亮度
        _Brightness("Brightness", Float) = 1
        //饱和度
        _Saturation("Saturation", Float) = 1
        //对比度
        _Contrast("Contrast", Float) = 1
    }
    SubShader
    {
        // No culling or depth
        Cull Off ZWrite Off ZTest Always

        Pass
        {
            CGPROGRAM
            #pragma vertex vert_img
            #pragma fragment frag

            #include "UnityCG.cginc"

            sampler2D _MainTex;
            half _Brightness;
            half _Saturation;
            half _Contrast;

            half4 frag (v2f_img i) : SV_Target
            {
                half4 col = tex2D(_MainTex, i.uv);

                //亮度
                half3 finalColor = col.rgb * _Brightness;
                //饱和度
                half lumin = Luminance(col.rgb);
                finalColor = lerp(lumin, finalColor, _Saturation);
                //对比度
                finalColor = lerp(0.5, finalColor, _Contrast);

                return half4(finalColor, col.a);
            }
            ENDCG
        }
    }
}

```

```csharp
float3 RGBToHSV(float3 c)
{
    float4 K = float4(0.0, -1.0 / 3.0, 2.0 / 3.0, -1.0);
    float4 p = lerp(float4(c.bg, K.wz), float4(c.gb, K.xy), step(c.b, c.g));
    float4 q = lerp(float4(p.xyw, c.r), float4(c.r, p.yzx), step(p.x, c.r));

    float d = q.x - min(q.w, q.y);
    float e = 1.0e-10;

    return float3(
        abs(q.z + (q.w - q.y) / (6.0 * d + e)),
        d / (q.x + e),
        q.x
    );
}

float3 HSVToRGB(float3 c)
{
    float4 K = float4(1.0, 2.0 / 3.0, 1.0 / 3.0, 3.0);
    float3 p = abs(frac(c.xxx + K.xyz) * 6.0 - K.www);

    return c.z * lerp(K.xxx, saturate(p - K.xxx), c.y);
}
```

# 碎屏效果
+ `_ScreenParams.x` ：当前屏幕或渲染目标的宽度
+ `_ScreenParams.y` ：当前屏幕或渲染目标的高度
+ `half2 glassMask = step(0.005, abs(glassNormal.xy));`：我们不希望屏幕整体全部发生偏移，否则屏幕上的画面也会发生移动。因此我们可以利用法线贴图人为计算出一张蒙版贴图，也就是只有法线贴图中有信息的位置我们才允许发生法线偏移。

```csharp
Shader "Hidden/CS05/BrokenGlass"
{
    Properties
    {
        [HideInInspector]_MainTex ("Texture", 2D) = "white" {}
        _GlassMask("Glass Mask", 2D) = ""{}
        _GlassColor("Glass Color", Range(0,10)) = 1
        _GlassNormal("Glass Normal", 2D) = ""{}
        _Distort("Distort", Range(0,5)) = 0
    }
    SubShader
    {
        // No culling or depth
        Cull Off ZWrite Off ZTest Always

        Pass
        {
            CGPROGRAM
            #pragma vertex vert_img
            #pragma fragment frag

            #include "UnityCG.cginc"

            sampler2D _MainTex;
            float4 _MainTex_ST;
            float4 _MainTex_TexelSize;
            sampler2D _GlassMask;
            float4 _GlassMask_ST;
            float4 _GlassMask_TexelSize;
            half _GlassColor;
            sampler2D _GlassNormal;
            float _Distort;

            half4 frag (v2f_img i) : SV_Target
            {
                //拉伸比转换
                float screenAspect = _ScreenParams.x / _ScreenParams.y;
                float glassAspect = (1 / _GlassMask_TexelSize.x) / (1 / _GlassMask_TexelSize.y);
                float2 uv_Glass = i.uv * _GlassMask_ST.xy + _GlassMask_ST.zw ;
                uv_Glass.x = uv_Glass.x * screenAspect / glassAspect;

                //玻璃采样
                half3 glassOpacity = tex2D(_GlassMask, uv_Glass).r;

                //获取法线
                half3 glassNormal = UnpackNormal(tex2D(_GlassNormal, uv_Glass));

                //扭曲uv
                half2 glassMask = step(0.005, abs(glassNormal.xy));
                half2 uv_Distort = i.uv + glassNormal.xy * _Distort * glassMask;

                half4 col = tex2D(_MainTex, uv_Distort);
                half3 finalColor = col.rgb;
                finalColor = lerp(finalColor, _GlassColor, glassOpacity);

                return half4(finalColor, col.a);
            }
            ENDCG
        }
    }
}
```

## 双重模糊
双重模糊指的是在使用其他模糊方式时，进行“**降采样 + 模糊 + 升采样**”的方式，即先进行`_Iteration`次模糊，模糊的同时对画面进行降采样，然后再进行`_Iteration`次模糊，但模糊的同时是对画面进行升采样，这样一来，多次缩小和放大会让高频细节不断丢失，只保留低频轮廓，我们就能避免其他模糊方式可能会导致的虚影以及像素化的问题；同时由于添加了降采样和升采样两步，因此我们可以通过更少的迭代次数获得更平滑的模糊结果，性价比极高。

# Bloom
传统的Bloom效果是通过图片的亮度作为阈值来提取亮部的，这里的“亮部”指的是人眼感知的亮部，但是由亮度公式可知，RGB三种颜色对应的系数分别为`0.22, 0.707, 0.071`，这就意味着画面中的蓝色部分的亮度最不容易被提取，从而导致画面色彩的变化。因此，我们将直接使用当前像素颜色分量中的最大值作为阈值判断的标准，以此来保证画面中高饱和的彩色光不会丢失。

```csharp
half4 color = tex2D(_MainTex, i.uv);
float br = max(max(color.r, color.g), color.b);
//不能使用saturate 此处使用的是HDR图像
br = max(0.0f, (br - _Threshold)) / max(br, 0.000001);
color.rgb *= br;
return color;
```

这样一来，当前像素最亮的颜色分量经过上述处理后，提取出的亮度正好就是`br - _Threshold`，并以`(br - _Threshold) / br`作为权重提取剩下两个分量。

同时，我们使用双重模糊的方式对光晕部分进行模糊，同时我们在升采样的过程中还需要将降采样时的`RenderTexture`传递给`Shader`，将其与当前画面相加，以此来防止画面过于模糊。
