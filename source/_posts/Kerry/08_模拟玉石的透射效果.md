# 背光效应
我们在观察一块玉石的时候，往往会将其正对着光源从其背面观察，以此来观察玉石的内部，我们的**视线方向越接近光源方向，透射效果就越亮**，因此，我们需要使用到**光源方向与视角方向之间的夹角**作为背光的权重：

`fixed3 backlight_color = _LightColor0 * albedo * saturate(dot(viewDir, -lightDir));`

{% note danger %}
注意`viewDir`指向的是摄像机，`lightDir`指向的是光源，其夹角为钝角，不好使用余弦值计算，因此我们将其中一个向量进行反向。

{% endnote %}

但是这样计算出来的结果显然只和光源方向与视角方向有关，与物体表面法线无关，因此渲染出来的物体的背光面看起来像一块平面，同时视角方向变化时，物体的亮暗变化也不是很明显，因此我们将点乘结果做**乘方**运算来提高对比度，同时将**光源方向与物体表面法线方向相加，即可模拟出物体的轮廓**：

```csharp
half3 backDir = -normalize(lightDir + worldNormal * _Disort);
fixed3 backlight_color = _LightColor0 * albedo * saturate(pow(saturate(dot(viewDir, backDir)), _Power));
```

{% note danger %}
注意我们需要给`dot`也加上`saturate`，否则在乘方运算的时候有可能出现给负数开根号的情况。

{% endnote %}

# 厚度图
上述得到的结果显然之和光源方向、视角方向、以及物体表面法线有关，但实际上结果还应当与物体的厚度有关：**光线穿越较厚的部分之后，显然剩余的光强更弱，因此我们需要使用厚度图**。

厚度图中的厚度指的是**沿着物体表面法线的反方向发出一条射线，直到碰撞到物体的背面**，并计算两点之间的距离，最终映射到$[0,1]$区间，越白越厚，越黑越薄。

```csharp
//透射光
half thickness = 1 - tex2D(_ThicknessMap, i.uv).r;
fixed3 backlight_color = _LightColor0 * albedo * saturate(pow(saturate(dot(viewDir, backDir)), _Power)) * thickness;
```

# 反射
玉石表面显然会反射周围的环境，这里我们使用立方体贴图以及简易菲涅尔边缘光来实现：

```csharp
//反射
half3 reflectDir = reflect(-viewDir, worldNormal);
//旋转
half s,c;
sincos(radians(_Rotate), s, c);
half2x2 rotate = half2x2(c, -s, s, c);
reflectDir.xz = mul(rotate, reflectDir.xz);
fixed3 reflect_color = DecodeHDR(texCUBE(_CubeMap, reflectDir), _CubeMap_HDR);
//边缘光
half fresnel = 1 - saturate(dot(viewDir, worldNormal));
reflect_color *= fresnel;
```

# 补光
现在物体没有被照亮的部分太黑了，因此我们将在补充漫反射的基础上再人为增加一种颜色来提亮，同时由于物体整体看上去过于透明，因此我们将引入**天光AO**作为不透明度参数进行调节：

```csharp
//漫反射
fixed3 diffuse = _LightColor0 * albedo * saturate(dot(lightDir, worldNormal));
UNITY_LIGHT_ATTENUATION(atten, i, i.worldPos);
//天光AO 以此来调整不透明度
half skylight_color = _Color * (dot(worldNormal, half3(0,1,0)) + 1) * 0.5;
//补光
fixed3 final_diffuse = diffuse + skylight_color * _Opacity + _AddColor.rgb;
```

{% note danger %}
天光AO：即法线方向越接近垂直向上，物体表面越亮。

{% endnote %}

            #include "AutoLight.cginc"

            {

            {
                v2f o;

                o.uv = TRANSFORM_TEX(v.texcoord, _MainTex);
                o.worldPos = mul(unity_ObjectToWorld, v.vertex);
                o.worldNormal = normalize(UnityObjectToWorldNormal(v.normal));
                TRANSFER_SHADOW(o)
                return o;

                float3 reflect_color = DecodeHDR(texCUBE(_CubeMap, reflectDir), _CubeMap_HDR);
                //边缘光

                reflect_color *= fresnel;

                float3 final_color = final_diffuse + reflect_color + backlight_color;

                return float4(final_color , 1);
            }
